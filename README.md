# Hex Purple Theme

![VS Code](https://img.shields.io/badge/VS_Code-007ACC?style=for-the-badge&logo=visualstudiocode&logoColor=white)
![Theme](https://img.shields.io/badge/Theme-purple?style=for-the-badge)

A dark VSCode theme with purple accents and vibrant syntax highlighting.

![Theme Preview](preview.png)

## Color Palette

- **Keywords** (`const`, `return`, `function`): `#C792EA` - Purple
- **Variables**: `#82AAFF` - Light Blue
- **Parameters/Props**: `#FFCB6B` - Yellow
- **Strings**: `#F07178` - Pink
- **Comments**: `#697098` - Gray/Violet (italic)
- **Function Names**: `#bdcff5` - Blue
- **Control Flow** (`if`, `async`, `await`): `#ff5e2d` - Orange

## Installation

### Local Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/hex-purple.git
   ```

2. Copy the theme folder to your VSCode extensions directory:
   ```bash
   cp -r hex-purple ~/.vscode/extensions/
   ```

3. Restart VSCode or reload the window:
   - Press `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows/Linux)
   - Type "Reload Window" and press Enter

4. Activate the theme:
   - Press `Cmd+K Cmd+T` (Mac) or `Ctrl+K Ctrl+T` (Windows/Linux)
   - Select "Hex Purple" from the list

## Manual Installation (Alternative)

If you prefer, you can also:

1. Open VSCode
2. Press `F5` while having this project open
3. A new VSCode window will open with the extension loaded
4. Select the theme with `Cmd+K Cmd+T`

## Development

To modify the theme:

1. Edit `themes/hex-purple-color-theme.json`
2. Reload VSCode to see changes
3. Adjust colors as needed
